# sally-copy-paste 설치 안내

macOS 클립보드 매니저입니다. 복사한 텍스트·이미지·파일을 모아두고, `Cmd+Shift+V`로 골라서 붙여넣습니다. (Intel·Apple Silicon 공용 유니버설 빌드)

## 1. 설치
1. 받은 `sally-copy-paste_0.1.0_universal.dmg` 를 더블클릭
2. 열린 창에서 **sally-copy-paste 앱을 Applications 폴더로 드래그**

## 2. 첫 실행 (중요 — 서명 안 된 앱)
Apple 공증을 받지 않은 빌드라, 처음에 macOS가 실행을 막습니다. 아래 중 하나로 한 번만 풀어주면 됩니다.

**방법 A (권장)**
- `응용 프로그램`에서 **sally-copy-paste 아이콘 우클릭 → 열기** → 경고창에서 **"열기"** 클릭
- (그냥 더블클릭하면 막히지만, 우클릭→열기는 통과됩니다. 최초 1회만)

**방법 B — A로 안 되거나 "손상되었기 때문에 열 수 없습니다"가 뜰 때**
- 터미널에 아래 한 줄 실행:
  ```
  xattr -dr com.apple.quarantine /Applications/sally-copy-paste.app
  ```
- 그 후 앱 실행

## 3. 권한 허용 (붙여넣기 자동화)
선택한 항목을 자동으로 붙여넣으려면 **손쉬운 사용** 권한이 필요합니다.
- **시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용**
- 목록에서 `sally-copy-paste` **켜기** (처음 붙여넣기 시도 시 권한 요청 팝업이 뜨기도 합니다)

## 4. 사용법
- 평소처럼 복사(`Cmd+C`) 하거나 Finder에서 파일 복사 → 자동으로 기록됨
- **`Cmd+Shift+V`** 또는 **메뉴바의 클립보드 아이콘 클릭** → 기록 팝업
- `↑`/`↓` 이동, `Enter` 또는 클릭 → 선택 → 직전 앱에 자동 붙여넣기
- 검색창에 입력하면 기록 필터링
- 항목 선택 후 `Delete`(또는 `Cmd+Delete`)로 삭제
- `Esc` 또는 바깥 클릭으로 팝업 닫기
- 종료: 메뉴바 아이콘 **우클릭 → 종료**

## 참고
- 텍스트·이미지·파일 모두 지원합니다.
- 기록은 본인 맥에만 로컬 저장됩니다(`~/Library/Application Support/com.sally.copypaste`).
- 앱 아이콘이 한동안 예전 것으로 보이면 로그아웃/재로그인하면 갱신됩니다.
